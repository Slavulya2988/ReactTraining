SPA - singlt page aplication - одностраничное приложение с действиями без перезагрузки страници
Плюси:
1. Деклатирность - что необхідно отримита в ітоги
2. Основан на компонентах
3. Еко-система реакта, в том числа React Native для сторвення мобільних додатків

Сборка проекта начинается с команди:
npx create-react-app my-app
cd my-app
npm start

Далее для роботи с препроцесором SASS необхідно додати библиотеку
npm i sass --save

Заново запускаем сервер
npm start
Теперь замість css можна використовувати scss
